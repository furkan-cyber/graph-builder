# API Reference

Graph Builder exposes a small root surface and a few focused subpath adapters.

## Root exports

- `graphBuilder`
- `createGraphBuilder`
- `extractText`
- `extractMarkdown`
- `buildGraph`
- `analyzeGraph`
- `serializeGraph`
- `generateReport`
- `generateWiki`
- `generateHtml`
- `updateGraph`
- `loadGraphBuilderResult`

## Subpath exports

- `@cognipeer/graph-builder/node`
- `@cognipeer/graph-builder/adapters/to-markdown`
- `@cognipeer/graph-builder/adapters/openai-compatible`
- `@cognipeer/graph-builder/adapters/memory`
