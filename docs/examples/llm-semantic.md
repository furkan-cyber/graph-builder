# LLM Semantic

Command:

```sh
npm run examples:llm
```

Environment variables:
- `OPENAI_API_KEY` or `GRAPH_BUILDER_OPENAI_API_KEY`
- Optional: `GRAPH_BUILDER_OPENAI_MODEL`
- Optional: `GRAPH_BUILDER_OPENAI_BASE_URL`

This example runs a real OpenAI-compatible semantic enrichment pass and writes an HTML visualization file.
