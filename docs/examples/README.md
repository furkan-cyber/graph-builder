# Examples

The repository includes runnable example scripts. Use these docs pages to decide which one to run first.

## Start here

- `examples:from-texts` for the smallest possible in-memory flow.
- `examples:provider` for integrating a custom source system.
- `examples:path` for local-file ingestion.
- `examples:llm-mock` for semantic enrichment without a real API key.
- `examples:llm` for the real OpenAI-compatible flow.
